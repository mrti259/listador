# listador
Este script permite la creación y organización de numerosas listas, bajo distintas categorías en diferentes archivos.

Cuando lo ejecutes por primera vez, se creará una carpeta ```archivos/``` en la que se guardaran
los distintos archivos que crees. Te pedirá el nombre del archivo al que quieras ingresar.
Si no existe, se creará uno nuevo.
Este archivo JSON almacena un diccionario que guardará las listas que quieras hacer.

Permite crear, eliminar, copiar o renombrar listas, y agregarles o eliminarles elementos.



Ej.:
```
archivos/libros.json
archivos/peliculas.json
archivos/musica.json
```
